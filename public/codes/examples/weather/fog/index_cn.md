#### fog 的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| enable | 开启/关闭雾天效果 | boolean | false           |
| start  | 最近可视距离      | number  | 0.1             |
| end    | 最远可视距离      | number  | 100             |
| color  | 雾的颜色          | color   | [0.9, 0.9, 0.9] |
