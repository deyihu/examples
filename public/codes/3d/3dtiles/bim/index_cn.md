#### Geo3DTilesLayer的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| services | 服务对象的数组，可包含多个服务 | Array | null           |
| url | 单个服务的地址 | String | null           |
| maximumScreenSpaceError  | 最大屏幕空间误差, 清晰度可以接受的情况下，推荐把这个值设得越大越好，性能会越好     | Number  | 1             |
| heightOffset  | 高度偏移量     | Number  | 0             |
