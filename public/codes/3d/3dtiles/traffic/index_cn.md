#### TrafficScene的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| carsNumber | 场景内车辆的数量 | Number | 100           |
| generateTraffic  | 生成交通路线，参数为坐标串数组     | 方法  |             |
|  addTo | 将场景添加到groupgllayer     | 方法  |             |
| setSymbols  | 设置车辆的模型样式    | 方法  |             |
