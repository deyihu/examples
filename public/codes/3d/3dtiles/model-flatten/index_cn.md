#### FlatInsideMask 的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| coordinates | 压平区域的范围坐标 | Array.<Array> \| Array.<Coordinate> |            |
| flatHeight  | 压平的高度      | Number  |       0      |

#### 方法 : setFlatHeight(height)
_设置ColorMask的高度范围_
| 参数名 | 说明 | 类型 | 默认值 |
| ------------- |---------- |-------------|--------- |
| `height`     | 压平的高度 | Number | 0 |
