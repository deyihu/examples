#### ColorMask 的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| symbol | ColorMask的符号对象，可参考maptalks.Polygon符号 | Object |            |
| heightRange  | 高度范围，分别为最大高度和最小高度      | Array  |             |

#### 方法 : setHeightRange(heightRange)
_设置ColorMask的高度范围_
| 参数名 | 说明 | 类型 | 默认值 |
| ------------- |---------- |-------------|--------- |
| `heightRange`     | 高度范围数组 | Array | 无 |
