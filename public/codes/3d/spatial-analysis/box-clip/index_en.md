#### BoxClipMask properties

| Properties   | Description              | Type    | Default          |
| ------ | ----------------- | ------- | --------------- |
| position | the box's position | Coordinate | null           |
| height  | box's height     | Number  | null            |
|  width | box's width     | Number  | null            |
| length  | box's length     | Number  | null            |
