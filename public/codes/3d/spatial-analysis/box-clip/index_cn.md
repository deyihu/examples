#### BoxClipMask的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| position | 盒子的位置坐标 | Coordinate | null           |
| height  | 盒子的高度     | Number  | null            |
|  width | 盒子的宽度    | Number  | null            |
| length  | 盒子的长度     | Number  | null            |
