#### FloodAnalysis 的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| boundary | 水淹区域的坐标范围 | Array | 无          |
| waterHeight  | 水面海拔高度，单位米      | number  | 0             |
| waterColor | 水颜色 | Array | [0.1451, 0.2588, 0.4863]           |
| waterOpacity  | 水颜色透明度      | number  | 1             |
