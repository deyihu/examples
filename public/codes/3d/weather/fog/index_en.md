#### Fog properties

| Properties | Description        | Type    | Default         |
| ---------- | ------------------ | ------- | --------------- |
| enable     | Enable/disable fog | boolean | false           |
| start      | Start distance     | number  | false           |
| end        | End distance       | number  | false           |
| color      | Fog color          | color   | [0.9, 0.9, 0.9] |
